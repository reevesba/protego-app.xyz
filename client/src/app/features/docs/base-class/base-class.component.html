Base

Base class that is inherited by the majority of classes in Protego.

This base class allows us to handle the following tasks in a uniform manner:

  • Saving and loading models to and from file.

Parameters

  • model : Union[base.BatchBase, base.BatchOnline]

    • SciKit Learn or River classifier.

Methods

Save a model as a pickled file.

Parameters: filename: string

Path and filename of pickled model to save.
Returns: None
