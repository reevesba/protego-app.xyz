DataLoader

Loads SQL injection payloads provided by Protego.

Contains benign and malicious samples collected from various sources including:

  • https://github.com/fuzzdb-project/fuzzdb/tree/master/attack/sql-injection/payloads-sql-blind
  • https://github.com/Morzeux/HttpParamsDataset/blob/master/payload_full.csv
  • https://www.kaggle.com/sajid576/sql-injection-dataset
  • https://www.kaggle.com/syedsaqlainhussain/sql-injection-dataset
  • https://archive-beta.ics.uci.edu/ml/datasets/sms+spam+collection
  • https://www.kaggle.com/uciml/sms-spam-collection-dataset
  • https://raw.githubusercontent.com/reevesba/sqli-detection/main/data/raw/payloads.txt?token=GHSAT0AAAAAABQOLK4TMQMWD7FHFC25V4FCYPWANXA
  • https://github.com/danielmiessler/SecLists/blob/master/Passwords/Common-Credentials/10-million-password-list-top-10000.txt
  • https://www.kaggle.com/henilvedant/sqlinjection-payload?select=raw2cleandata2.csv
