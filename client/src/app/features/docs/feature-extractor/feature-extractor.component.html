FeatureExtractor

Used to extract SQL injection payload features.

Extracted features include:

  • num_chars: number of characters.
  • num_words: number of words.
  • num_special_chars: number of special characters.
  • num_ticks: number of single quotes.
  • num_dashes: number of dashes.
  • num_commas: number of commas.
  • num_pipes: number of pipe operators.
  • num_equals: number of equals signs.
  • num_keywords: number of SQL keywords.
  • num_parens: number of parenthesis.
  • avg_word_len: average word length.
  • num_white_spaces: number of white spaces.
  • num_comments: number of SQL comments.
  • label: classification.

Parameters

  • None

Methods

info transform
